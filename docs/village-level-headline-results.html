3.1 People using DSW/ILC water points

3.1.1 Prepare data

To calculate the number of people using water points with DSW or ILC, we use self-reported data from the household census on what is the households primary water point. This information is combined with data from the water point census to determine whether these water points are served by DSW or ILC.

We start by restricting the household census data to households using water points with DSW or ILC located inside the sampled villages. We also exclude households using ILC in Uganda because we did not design the survey to be representative of this population.

Code 
users_data <-
  hh_census %>%
  dplyr::filter(
    wp_intervention_type != "Non-program",
    wp_invillage,
    !(sample_group != "ILC" & wp_intervention == "ILC")
  )

The relevant variable to calculate the number of people using water points with DSW or ILC is the number of people living in households that report their primary water source has DSW or ILC. Due to a coding mistake, this variable was not collected in the first few days of data collection in Uganda. Therefore, we need to impute values for the households where this information is missing.

Code 
users_data <-

To account for the villages where all the water points with ILC were out of order, we add them to the data, but with zero households using water points with ILC.

Since the unit of reference for our analysis is the village, we calculate the number of people whose primary water source has DSW or ILC by adding the number of household members across all such households in a village.

Code 
users_village <-
  users_data %>%

    response_rate = unique(response_rate),

  mutate(

3.1.2 Sanity check

Here we just calculate the total number of people without CIs. The point estimate should be the same as the one calculated with a canned function.

users_village %>%

  select(country, wp_intervention, outcome,
         mean, ci, hhs, villages) %>%
  pivot_wider(
    id_cols     = c(country, wp_intervention),
    names_from  = outcome,  
    values_from = c(mean, ci, hhs, villages),
    names_glue  = "{outcome}_{.value}"
  )

tab <- users_sum %>%
  left_join(cl_sum, by = c("country", "wp_intervention")) %>%
  arrange(factor(country, levels = c("Malawi", "Uganda")),
          factor(wp_intervention, levels = c("DSW", "ILC")))

fmt_int <- function(x) format(round(x), big.mark = ",", trim = TRUE)

rows <- character()

for (i in seq_len(nrow(tab))) {
  r <- tab[i, ]

  if (i == 1 || tab$country[i] != tab$country[i - 1]) {
    if (i > 1) rows <- c(rows, "\\addlinespace")
    rows <- c(
      rows,
      glue("\\textbf{{{r$country}}} &  &  &  &  &  &  &  & \\\\")
    )
  }

  prog <- r$wp_intervention

  tcr_mean <- r$disctcr_02_mean
  tcr_ci   <- r$disctcr_02_ci
  fcr_mean <- r$discfcr_02_mean
  fcr_ci   <- r$discfcr_02_ci

  line1 <- glue(
"{prog} & {fmt_int(r$vil_hh)} & {fmt_int(r$hhs_hh)} & {fmt_int(r$vil_country)} & {fmt_int(r$people_tot)} & {fmt_int(r$disctcr_02_villages)} & {fmt_int(r$disctcr_02_hhs)} & {tcr_mean} & {fcr_mean}\\\\"
  )

  line2 <- glue(
" &  &  &  & ({fmt_int(r$people_lb)}, {fmt_int(r$people_ub)}) &  &  & {tcr_ci} & {fcr_ci}\\\\"
  )

  rows <- c(rows, line1, line2)
}

writeLines(
  rows,
  here("../output/tables/headlines-vil.tex")
